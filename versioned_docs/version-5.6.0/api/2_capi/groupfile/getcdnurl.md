# GetCDNUrl

Return CDN url for file for clients that cannot handle redirect.

<PartialServop service_name="groupFile" operation_name="GET_CDN_URL" />

## Method Parameters
Parameter | Description
--------- | -----------
groupId | The id of the group.
fileId | The id of the file.
