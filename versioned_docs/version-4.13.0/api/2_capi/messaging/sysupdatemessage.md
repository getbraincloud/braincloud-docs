# SysUpdateMessage
## Overview
:::tip
This method is available in Cloud Code scripts only.
:::

## Method Parameters
Parameter | Description
--------- | --------------------------------
msgId | The id of the message to update.
msgVer | The version of the message.
content | The new content to write to the message.
